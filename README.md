# 💻 IT Support Labs

This repository contains hands-on IT Support and Service Desk troubleshooting labs completed in a Windows 10 virtual environment.

Each lab simulates a real-world IT support incident and follows a structured troubleshooting process from user complaint to issue resolution.

## 🎯 Purpose

The purpose of this repository is to strengthen practical IT Support skills by solving common technical issues and documenting each troubleshooting process in a professional manner.

## 🛠 Skills Practiced

- Windows 10 Troubleshooting
- Desktop Support
- Hardware Troubleshooting
- Incident Management
- Root Cause Analysis (RCA)
- Task Manager Analysis
- Windows Performance Monitoring
- Basic Network Troubleshooting
- Remote Support
- Ticket Documentation
- IT Service Desk Best Practices

## 💻 Lab Environment

- VirtualBox
- Windows 10
- 2 GB RAM
- 1 CPU Core
